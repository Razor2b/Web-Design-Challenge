Summary Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OPenWeatherMap API to assemble a dateset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. Latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humitity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends adn correlations witnessed.
